⚙ Google Sheets — Zone URLs

In Google Sheets: File → Share → Publish to web → choose the tab → format CSV → Publish → copy the URL.
Add as many zones as you need. Each zone becomes its own tab in the dashboard.

Expected columns: Zone, Date, # Orders, WH, Active Riders, UTR, On Time, DT, Accept Time, Transport Fail, Late Login, No Show, Break Hours.

⏳
Update numbers from the zone sheet

Open any zone tab in the sheet (Mohandiseen / Hadayek el Qobba / Shobra), select the rows with their headers, copy them, and paste here. It picks up the latest two days automatically. You can paste all three zones together or one at a time. Or drag a CSV/text file and drop it in the box.

Edit mode is on — change the values directly in the table and everything recalculates.

How to read the colors

Green = improved (moved the right way for that metric) Red = dropped Side bar = strong move that needs attention
OK ✓ 0 metrics dropped Watch 1–2 Needs action 3 or more
Source: the zone sheet (daily numbers) + the offices sheet "3PL WH" for drilling into a working-hours drop.
At office level only working hours are available right now — the other metrics are still at zone level.
